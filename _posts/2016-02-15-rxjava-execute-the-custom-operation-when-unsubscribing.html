The following example will show how to perform the custom operation when unsubscribing

{% highlight wl linenos %} Service service = new Service(); Observable ob = Observable.create(subscriber -> { subscriber.onNext("test"); service.register(); subscriber.add(Subscriptions.create(() -> { // do custom operation service.unregister(); })); } ); {% endhighlight %}
