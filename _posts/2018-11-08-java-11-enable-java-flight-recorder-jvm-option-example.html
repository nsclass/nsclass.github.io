-XX:StartFlightRecording=\
        disk=true, \
        dumponexit=true, \
        filename=recording.jfr, \
        maxsize=1024m,\
        maxage=1d

Checking running JFR with jcmd

% jcmd PID JFR.check name=1

Dump JFR now

%jcmd PID JFR.dump name=1 filename=dump.jfr
