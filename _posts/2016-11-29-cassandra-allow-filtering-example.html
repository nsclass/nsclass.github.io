[code lang="c"]
create table User(
email text,
name text,
desc text,
user_id uuid
PRIMARY KEY((email), name, user_id)
) WITH CLUSTERING ORDER BY (name DESC, user_id ASC);
[/code]

Allow Filtering can only work with the clustering key.

[code lang="c"]
select * from user where name = 'Test' allow filtering;
[/code]
