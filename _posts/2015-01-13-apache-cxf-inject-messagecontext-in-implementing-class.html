It is common to get a Servlet context on implementing WEB API. Apache CXF has a little different approach to get this instance. The following code show how to get ServletContext instance for RS.

{% highlight wl linenos %} public interface IServiceInterface { void SampleFunction(); } public class ImplementClass implement IServiceInterface { @Context private org.apache.cxf.jaxrs.ext.MessageContext messgeContext; public void SampleFunction() { ServletContext servletContext = messgeContext.getServletContext(); } } {% endhighlight %}
