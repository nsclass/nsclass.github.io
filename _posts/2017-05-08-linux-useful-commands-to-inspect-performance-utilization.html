Memory utilization
[code lang="C"]
vmstat 1
[/cpde]

Disk utilization
[code lang="C"]
iostat -xm 5
[/code]

Network utilization tool
[code lang="C"]
nicstat
[/code]

Network connection status
[code lang="C"]
netstat
[/code]

Note, Windows has the following equivalent command for above purpose
[code lang="C"]
typeperf
[/code]
