List modules
{% highlight wl linenos %} $ java --list-modules $ java --list-modules java.base {% endhighlight %}

Diagnostic module dependencies
{% highlight wl linenos %} $ java -Xdiag:resolver -p mods [module name]/[class package path] $ jar --file [jar path] --print-module-descriptor {% endhighlight %}
