List modules
[code lang="c"]
$ java --list-modules
$ java --list-modules java.base
[/code]

Diagnostic module dependencies
[code lang="c"]
$ java -Xdiag:resolver -p mods [module name]/[class package path]
$ jar --file [jar path] --print-module-descriptor
[/code]
