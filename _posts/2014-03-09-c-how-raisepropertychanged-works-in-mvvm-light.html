MVVM light provides an easy way of the strong typed RaisePropertyChanged with lambda. The following code shows how this can find the correct string for member of ViewModel.

{% highlight wl linenos %} public void RaisePropertyChanged(Expression> property) { PropertyChanged(this, new PropertyChangedEventArgs(property.GetMemberInfo().Name); } {% endhighlight %}

so you can use the following code safely.
{% highlight wl linenos %} public class ViewModel : ViewModelBase { private string _name; public string ProductName; { get { return _name; } { set { _name = value; RasePropertyChanged(() => ProductName); } } } {% endhighlight %}
