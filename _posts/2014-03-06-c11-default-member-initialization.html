You can initialize the member variable as shown the following in C++11. It will save a lot of typing when you have many constructors.

{% highlight wl linenos %} enum class BuildingType { None, Resident, Commercial, }; class Building { BuildingType m_type { BuildingType::None }; std::string m_address { "None" }; int m_roomCount { 0 }; std::string m_name; Building(std::string name) : m_name(std::move(name)) { } }; {% endhighlight %}
