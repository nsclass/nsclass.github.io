Java 8
-XX:+PrintGCDetails
-XX:+PrintReferenceGC
-XX:+PrintTenuringDistribution
-Xloggc:
-XX:+PrintGCTimeStamps

Java 9
-Xlog:gc*
,gc+ref=debug
,gc+age=trace
