NavigableMap collection on Java provide methods to return the view of Map on sorted order of key. The most common methods are headMap, tailMap and subMap.

headMap example.
{% highlight wl linenos %} NavigableMap navigableMap = new TreeMap<>(); navigableMap.put("1", "1"); navigableMap.put("2", "2"); navigableMap.put("3", "3"); navigableMap.put("4", "4"); SortedMap resultViewMap = navigableMap.headMap("3"); resultViewMap.entrySet().stream().forEach(entry -> System.out.println(String.format("key: %s, value: %s", entry.getKey(), entry.getValue()))); {% endhighlight %}

output
{% highlight wl linenos %} key: 1, value: 1 key: 2, value: 2 {% endhighlight %}

If inclusive is true on calling headMap("3", true), it will be
{% highlight wl linenos %} key: 1, value: 1 key: 2, value: 2 key: 3, value: 3 {% endhighlight %}

tailMap example.
{% highlight wl linenos %} NavigableMap navigableMap = new TreeMap<>(); navigableMap.put("1", "1"); navigableMap.put("2", "2"); navigableMap.put("3", "3"); navigableMap.put("4", "4"); SortedMap resultViewMap = navigableMap.tailMap("3"); resultViewMap.entrySet().stream().forEach(entry -> System.out.println(String.format("key: %s, value: %s", entry.getKey(), entry.getValue()))); {% endhighlight %}

output
{% highlight wl linenos %} key: 3, value: 3 key: 4, value: 4 {% endhighlight %}

By default, inclusive is true on calling tailMap("3", true).

subMap example
{% highlight wl linenos %} NavigableMap navigableMap = new TreeMap<>(); navigableMap.put("1", "1"); navigableMap.put("2", "2"); navigableMap.put("3", "3"); navigableMap.put("4", "4"); SortedMap resultViewMap = navigableMap.subMap("2", "4"); resultViewMap.entrySet().stream().forEach(entry -> System.out.println(String.format("key: %s, value: %s", entry.getKey(), entry.getValue()))); {% endhighlight %}

output
{% highlight wl linenos %} key: 2, value: 2 key: 3, value: 3 {% endhighlight %}

By default, inclusive is true for fromKey, and inclusive is false for toKey on calling subMap("2", true, "4", false)

There are more available methods in NavigableMap
